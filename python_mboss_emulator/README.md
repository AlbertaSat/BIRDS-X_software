# python-serial-tester

A simple testing framework to act as the Mission Boss for the BIRDS-X JASPER development.

## Running
1. Install Python3.11 or similar.
2. Install dependencies with `python3.11 -m pip install -r ./requirements.txt`
3. Run the Python file with `python3.11 emulate_mboss.py`
4. Follow the GUI prompts, while looking at the terminal log.
